# Linux Virtual Machine Environment

## Objective
Build and maintain Linux virtual machine environments for cybersecurity training, networking, scripting, and offensive security lab exercises.

## Platforms Used
- Oracle VirtualBox
- Parrot Security OS
- Kali Linux
- Debian
- Arch Linux

## Skills Practiced
- Linux administration
- Virtualization
- Package management
- VM configuration
- Display troubleshooting
- Guest Additions installation
- Desktop environment management
- Terminal operations

## Tasks Performed
- Created Linux virtual machines
- Installed and configured Linux operating systems
- Installed VirtualBox Guest Additions
- Configured graphical desktop environments
- Troubleshot display and resolution issues
- Managed Linux packages and dependencies
- Configured networking and VPN connectivity

## Lessons Learned
This environment improved Linux troubleshooting capability, virtualization experience, system configuration familiarity, and command-line operational workflow development.
