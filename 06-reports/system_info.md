# Linux System Enumeration Script

## Objective
Create a Bash script that collects basic host system information from a Linux environment.

## Script Location
`02-bash-scripting/system_info.sh`

## Skills Practiced
- Bash scripting
- Linux enumeration
- Network interface review
- Process inspection
- System resource review
- Output filtering with pipes

## Commands Used

| Command | Purpose |
|---|---|
| `hostname` | Displays the system hostname |
| `whoami` | Displays the current user |
| `uname -r` | Displays the Linux kernel version |
| `ip addr` | Displays network interface information |
| `grep inet` | Filters IP address information |
| `df -h` | Displays disk usage in human-readable format |
| `free -h` | Displays memory usage in human-readable format |
| `ps aux` | Displays running processes |
| `head` | Limits output to the first few lines |

## What I Learned
This exercise demonstrated how Bash can automate Linux host enumeration and collect operationally relevant system information.
